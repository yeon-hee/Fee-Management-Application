## 👨‍💻 동아리 회비 관리 애플리케이션
<br>

### 👪 프로젝트 기간 및 참여 인원
- 2018.09 ~ 2018.11 개인 프로젝트

### 📄 개발 목적
- JAVA 언어를 이용한 Android 앱 개발 및 동아리 회비 관리 프로그램 구현 

### 🔧 사용 언어 및 라이브러리
- JAVA
- Android

### 💻 개발 환경
- OS : Window
- Tool : Android Studio

### 📋 구현 기능
- 회원 등록 : 이름을 입력하면 database에 멤버 추가
- 회원 정보 : database에 기록한 회원 정보 출력
- 회비 수납 : 이름과 금액을 입력 시 databse에 기록
- 미납자 : 해당 월에 대한 회비 미납자 출력
- 회비 합계 : 1년치 회비 수납 현황 출력
